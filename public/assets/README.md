# Portfolio image assets

Place optimized portfolio images here.

Recommended structure:
- `assets/cv_visual.webp` — hero/profile visual, 780×1020px, preferably < 250 KB.
- `assets/projects/<project-slug>-cover.webp` — optional project cover images, 1200px wide, preferably < 350 KB.

Export rule:
1. Keep original high-resolution files outside the deployed repository, for example in a private `source-images/` folder.
2. Deploy only compressed `.webp` or `.avif` files.
3. Use English lowercase filenames with hyphens, no spaces, no Chinese characters.
